Detailed Description

ProcessTool is the abstract class which is used to control the settings of a process with UI in a dialog box. A ProcessTool class consistent of a ProcessView member to display the processed image and a ProcessWrap class member to aintain the process related parameters and LUT data. ProcessTool is the bridge class between WorkView and the Process. The application use ProcessTool to access the specified process stuff.
