Detailed Description

The data of process parameters are depedent on individual process tool. Each tool will produce LUT data, which is maintained by this class. The ProcessCap keeps process's parameters and allocate memory space for LUT. The struct of a Process's parameter is defined by each Process and decalred in common.h.
