Detailed Description

wrap process parameters

The data of process parameters are depedent on individual process tool. Each tool will produce LUT data, which is maintained by this class. The ProcessWrap keeps process's parameters and allocate memory space for LUT. The struct of a Process's parameter is defined by each Process and decalred in common.h
